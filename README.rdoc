== NMC Call Log

This is a web application built for the Nursing Mother's Counsel of Oregon:

http://www.nursingmotherscounsel.org

There are 2 core functions of the app:
* Provide webhooks to Twilio to (1) Process an incoming call (2) Record the completion of the call
* Provide a mobile-friendly form to volunteers to capture the details of the completed call
